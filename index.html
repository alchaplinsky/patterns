Common patterns of markup and style

This guide is created to share and describe how common frontend tasks can be handled with HTML/CSS and to share best practices across developers all over the world.

This is a collection of interface patterns in HTML/CSS for frequently solved tasks. Don't spend much time on making up custom class name or finding the most flexible solution. Remember most of problems, you're cracking your head on right now, had been already solved by someone!

Whether you are looking for a solution or for nice class name for your UI components - you've come to right place!

This book can be used as a learning resource for Javascript training if you want, but please send a tweet recommending us to other people.

I hope this guide will be helpful and will give you new ideas.
